## Quick Example

```bash
curl -X POST https://api.fileloom.io/v1/pdf \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "templateId": "tpl_invoice_001",
    "templateData": {
      "customerName": "Acme Corp",
      "items": [
        {"name": "Widget", "quantity": 5, "price": 29.99}
      ]
    }
  }'
```
