## What is Fileloom?

Fileloom is a PDF generation API built for developers who need reliable, fast, and scalable document generation. Send HTML with dynamic data, get back a professionally formatted PDF.

**One endpoint. Endless possibilities.**
```bash
POST https://api.fileloom.io/v1/pdf/generate
```

## How It Works

<Steps>
  <Step title="Create a Template">
    Design your PDF using HTML and CSS in our visual editor, or send raw HTML via the API.
  </Step>
  <Step title="Send Data">
    Make an API request with your template ID and dynamic data (JSON).
  </Step>
  <Step title="Get Your PDF">
    Receive a URL to your generated PDF, ready to download, embed, or store.
  </Step>
</Steps>

## Use Cases

Fileloom powers document generation for:

- **Invoices & Receipts** — Dynamic line items, tax calculations, payment terms
- **Contracts & Agreements** — Merge customer data, e-signatures ready
- **Reports & Analytics** — Charts, tables, data visualizations
- **Certificates** — Event tickets, course completions, awards
- **Shipping Labels** — Barcodes, QR codes, address formatting

## Quick Example

Generate a PDF with a single API call:
```bash
curl -X POST https://api.fileloom.io/v1/pdf/generate \
  -H "X-API-Key: fl_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "htmlContent": "<h1>Hello, {{name}}!</h1><p>Welcome to Fileloom.</p>",
    "templateData": {
      "name": "World"
    }
  }'
```

**Response:**
```json
{
  "success": true,
  "data": {
    "fileId": "file_1234567890_abc123",
    "url": "https://storage.googleapis.com/fileloom/...",
    "filename": "document.pdf",
    "size": 12847,
    "processingTimeMs": 847
  }
}
```
